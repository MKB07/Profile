# Genetic Pedigree Builder

Repository: https://github.com/MKB07/pedigree-builder

## Overview

Genetic Pedigree Builder is an R/Shiny application for constructing, annotating and exploring genetic pedigrees.

It was built through self-directed programming learning, iterative testing and AI-assisted coding support. AI tools helped with documentation, debugging and implementation scaffolding, while the genetic concepts, feature choices and interpretation constraints were driven by clinical genetics experience.

## Why This Project Exists

Pedigrees are a central representation in genetic counseling. They help clinicians and patients understand family structure, inheritance patterns, risk and relevant relatives. Existing tools are often not flexible enough for exploratory or educational workflows, so this project was developed as a practical interface for genetic pedigree work.

## Main Features

- Interactive pedigree creation.
- Individual and phenotype annotations.
- Relationship exploration.
- Kinship and inbreeding-related calculations.
- Gene and GeneReviews exploration modules.
- Export and reload workflows.
- Defensive plot scaling for dense pedigrees and smaller screens.

## Technical Stack
